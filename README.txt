# writing-author-management

내용
글 추가, 삭제, 수정과 저자의 삭제, 수정, 추가가 가능하다.

파일
db.sql : db table생성과 

main.js : 프로그램의 메인

lib
author.js : 저자관련 코드 존재

db.js : mysql 연결을 위한 기본설정
(user, password 사용자의 정보 기입 필수)

template.js : html파일의 template이 존재하는 파일

topic : 글(주제)에 대한 코드 존재


필요한 라이브러리
npm install mysql
npm install http
